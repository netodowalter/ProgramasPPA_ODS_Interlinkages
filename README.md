# ProgramasPPA_ODS_Interlinkages
Classifica programas do PPA 2024-2027 conforme os ODS e a Metas ODS afetados. A partir de textos do programa isolados (justificativa, objetivo geral, descrição do problema, objetivos específicos) ou conjuntamente.

Para uso em Windows 11, Python 11 com Cuda (Nvidia RTX 3060+).
